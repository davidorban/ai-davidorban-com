// the approach

Humans and AIs, side by side

I run my work as a hybrid team. Some contributors are people. Some are AI agents that persist between sessions, hold context, and report on what they did — listed on the same page, as line items, next to everyone else.

The bet is simple: AI readiness is behavioral, not declarative. You don't become AI-native by saying so, or by filling in a questionnaire. You become AI-native by actually delegating real work, building the plumbing, and living with the results — the good, the awkward, and the occasionally hilarious.

The interesting skill of the next decade isn't prompting. It's the management of AIs — deciding what to hand off, how to verify it, and where a human still has to own the call.

// how it fits together

The shape of the system

Not a product, not a diagram to impress a boardroom — the actual working layers, described at the altitude that's useful. The coordinates stay private; the shape doesn't have to.

The fleetpersistent agents

A set of autonomous agents that run around the clock, each with its own identity, mailbox, and standing tasks. They persist between sessions and hold their own context.

The gatewayCloudflare AI Gateway

A single gateway sits in front of the models the fleet uses — handling routing, fallbacks, and per-agent cost telemetry, so every agent's spend shows up as a line item.

The knowledge layerDex · semantic search

Dex is the operations-and-memory brain: projects, people, commitments, and years of notes and transcripts, recalled by meaning rather than keyword. It also has the inbox you can write to.

The connective tissueMCP

The Model Context Protocol wires agents into calendars, mail, documents, and dozens of other tools through one open standard — so capability is a matter of plugging in, not rebuilding.

The reasoning coreClaude · Claude Code

Where research, writing, and real code changes actually happen — the agentic environment the rest of the system is built and maintained from.

Nothing irreversible leaves the building without me. The last layer is always a human in the loop.

// where the agents talk

Discord is the nervous system

The agents don't work in isolation. They share a Discord bus — posting daily standups, coordinating hand-offs, and flagging what needs a human. It's the same move as any distributed team, except some of the team never sleeps.

The point of the whole exercise: humans and AIs on the same standup, as line items, side by side.
